3. 주요 Actuator 엔드포인트
라우트 목록 조회:

GET /actuator/gateway/routes - 모든 라우트 정보 조회
GET /actuator/gateway/routes/{route_id} - 특정 라우트 상세 정보

라우트 관리:

POST /actuator/gateway/refresh - 라우트 새로고침
POST /actuator/gateway/routes/{route_id} - 라우트 추가
DELETE /actuator/gateway/routes/{route_id} - 라우트 삭제

필터 조회:

GET /actuator/gateway/globalfilters - 글로벌 필터 목록
GET /actuator/gateway/routefilters - 라우트 필터 목록
